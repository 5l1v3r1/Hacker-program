加密病毒(勒索
=
     编译环境VS2019 X64
	 
	 

	 
使用方法：
-
     clon https://github.com/Ascotbe/Hacker-program.git
	 然后编译
	 TOP：禁止做非法操作，仅供逆向师傅参考流程（无壳无内存加密）。
	 目前实现功能：
	 1.扫描硬盘文件
	 2.修改EXE文件图标
	 3.复制自身到指定目录
	 4.下载文件
	 5.修改注册列表
	 6.修改壁纸
	 7.释放到桌面一个文件
	 8.加密C:\1564ww这个文件夹里的文件
	 9.释放后台监控
	 10.双击后后台运行不会跳出CMD窗口
	 11.提升权限
	 12.过DEP保护
	 13.硬盘锁
	 14.自删除
	 

由于病毒破坏性很强后面具有破坏性的功能就不上传了，只上传了EXE文件。
